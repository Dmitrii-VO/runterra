# Функционал вкладок приложения

Справочник по функционалу каждой вкладки мобильного приложения Runterra.

## Вкладки (BottomNavigationBar)

| Вкладка | Документ | Краткое описание |
|---------|----------|------------------|
| **Карта** | [map-tab.md](map-tab.md) | Территории, события, клубы на карте Yandex MapKit |
| **Пробежка** | [run-tab.md](run-tab.md) | GPS-трекинг, журнал тренировок, детали пробежек |
| **Сообщения** | [messages-tab.md](messages-tab.md) | Клубные чаты и тренер↔клиент 1:1 (реализованы), личные (заглушка) |
| **События** | [events-tab.md](events-tab.md) | Список, создание, участие, check-in, Swipe-to-run |
| **Профиль** | [profile-tab.md](profile-tab.md) | Личные данные, город, клубы, настройки, выход |

## Навигация

- Маршруты вкладок: `/map`, `/run`, `/messages`, `/events`, `/` (профиль)
- `StatefulShellRoute.indexedStack` сохраняет состояние каждой вкладки при переключении
