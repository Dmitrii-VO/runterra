# Вкладка «Карта»

Функционал вкладки «Карта» мобильного приложения Runterra.

## Назначение

Интерактивная карта города с отображением территорий (беговых зон), событий и клубов. Точка входа для навигации по беговой инфраструктуре города.

## Экраны и компоненты

### MapScreen

Основной экран вкладки. Использует Yandex MapKit.

**Параметры:**
- `showClubs` — при `true` после загрузки показывается bottom sheet со списком клубов города
- `focusLatitude`, `focusLongitude` — координаты для центрирования карты при открытии (например, при переходе с мини-карты события)

**Стартовая позиция:**
- Центр города из `CurrentCityService` (по `cityId` пользователя)
- Fallback: Санкт-Петербург (59.9343, 30.3351), zoom 12

## Отображаемые объекты

### Территории

- **Визуализация:** полигоны (при наличии `geometry`) или круги радиусом 500 м (fallback)
- **Цвета по статусу:**
  - CAPTURED — синий (захвачена клубом)
  - FREE — серый (нейтральная)
  - CONTESTED — жёлтый (оспариваемая)
  - LOCKED — тёмно-серый
- **Тап на территорию:** TerritoryBottomSheet — название, статус, клуб-владелец, кнопка «Подробнее» → `/territory/:id`

### События

- **Маркеры:** программные иконки (оранжевый круг с календарём)
- **Тап на маркер:** bottom sheet с EventCard — дата/время, место, организатор, кнопка «Подробнее» → `/event/:id`

## Действия пользователя

| Действие | Результат |
|----------|------------|
| Кнопка «Моё местоположение» | Центрирование карты на GPS-координатах пользователя |
| Кнопка «Найти клуб» (из навигации) | Переход `/map?showClubs=true` → bottom sheet со списком клубов |
| Тап по клубу в bottom sheet | Закрытие sheet, переход `/club/:id` |
| Тап на мини-карту события | Переход `/map?lat=..&lon=..` с центрированием на точке старта |

## Ограничения камеры

- Zoom ограничен границами города (`bounds` из конфига городов)
- При изменении позиции камеры проверяется, что центр остаётся в пределах города

## API

**GET /api/map/data**
- Query: `cityId` (обязательный), `clubId?`, `onlyActive?`
- Ответ: `MapDataDto` — `territories[]`, `events[]`

## Навигация

- Маршрут: `/map`
- Query: `showClubs`, `lat`, `lon` (для focus)
