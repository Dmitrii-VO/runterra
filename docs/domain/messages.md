# Messages — MVP (123.md)

## Назначение

Messages в MVP — не мессенджер, а:

- инструмент координации внутри клуба;
- канал сигналов (территории, тренировки);
- способ создать ощущение живого сообщества.

**Не в MVP:** личные чаты, соцсеть, лента новостей.

## Структура вкладок

| Вкладка       | Содержимое                          |
|---------------|-------------------------------------|
| **Город**     | Общий (городской) чат               |
| **Клубы**     | Список чатов клубов                 |
| **Уведомления** | Системные сообщения (in-app inbox) |

### Город

- Короткие текстовые сообщения, эмодзи, @клуб.
- Ограничения: без медиа, без личных ответов, без тредов, лимит 500 символов.
- Пишут: любой пользователь города, клубы от имени клуба (опц.).

### Клубы

- Отдельный чат на клуб. Доступ: участники; меркатели — только если записаны на тренировку.
- Текст + системные события (тренировка создана, территория захвачена, под угрозой, новый участник).
- Роли: лидер/модератор (удаление, закрепление), участник (только писать).

### Уведомления

- Не чат. Личный центр уведомлений: напоминание о тренировке, территория оспаривается/захвачена, приглашение в клуб, отклонение запроса.
- Только чтение, клик → экран (карта / клуб / тренировка), можно отметить «прочитано».

## Не в MVP

- Личные переписки, медиа, реакции (кроме простых эмодзи), треды/ответы, поиск, экспорт/пересылка.
