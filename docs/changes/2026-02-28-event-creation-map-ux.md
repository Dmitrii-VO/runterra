# Создание события с начальным типом и маркеры по zoom

**Дата:** 2026-02-28

## Описание

Создание события получает начальный тип из контекста (вкладка «Тренировка» / «Открытое событие»). На карте маркеры событий отображаются в зависимости от уровня zoom для лучшей читаемости.

## Mobile

### CreateEventScreen и маршрут создания

- Экран создания события принимает опциональный query-параметр `type`: `training` или `open_event`.
- Маршрут `/event/create` может вызываться с `?type=training` или `?type=open_event`.
- При переходе на создание с вкладки «События» контекст таба (тренировка / открытое событие) передаётся в маршрут, и форма открывается с уже выбранным типом.

### MapScreen — маркеры событий

- Отображение маркеров событий на карте сделано условным в зависимости от уровня zoom.
- При большом охвате (малый zoom) маркеры могут не отображаться или отображаться реже, чтобы не загромождать карту; при приближении — полный набор. Улучшает UX при работе с картой.

## Файлы

| Файл | Действие |
|------|----------|
| `mobile/lib/app.dart` | Маршрут создания с query `type` |
| `mobile/lib/features/events/create_event_screen.dart` | Приём и применение начального типа из query |
| `mobile/lib/features/events/events_screen.dart` | Передача типа при переходе на создание |
| `mobile/lib/features/map/map_screen.dart` | Условное отображение маркеров по zoom |
