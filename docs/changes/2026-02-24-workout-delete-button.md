# Добавление кнопки удаления тренировки (2026-02-24)

## Контекст
Пользователи не могли найти способ удалить созданную тренировку. Хотя в списке тренировок была реализована функция swipe-to-delete, она не была очевидной. Требовалась явная кнопка удаления на экране редактирования тренировки.

## Выполненные изменения

### 1. Локализация
- В файлы `app_ru.arb` и `app_en.arb` добавлен новый ключ `workoutDeleteAction` ("Удалить" / "Delete").
- Обновлены сгенерированные файлы локализации (`app_localizations.dart`, `app_localizations_ru.dart`, `app_localizations_en.dart`) для поддержки нового ключа.

### 2. Мобильный клиент
- В `WorkoutFormScreen` (`mobile/lib/features/trainer/workout_form_screen.dart`) добавлен метод `_delete()`, который:
  - Отображает диалог подтверждения удаления.
  - При подтверждении вызывает `ServiceLocator.workoutsService.deleteWorkout()`.
  - Обрабатывает ошибки (например, `workout_in_use`, если тренировка привязана к будущим событиям), показывая локализованное сообщение.
  - При успешном удалении закрывает экран и возвращает `true`.
- В интерфейс формы добавлена кнопка «Удалить» (красного цвета), которая отображается только при редактировании существующей тренировки. Кнопка расположена ниже кнопки «Сохранить».
