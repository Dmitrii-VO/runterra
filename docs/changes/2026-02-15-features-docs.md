# Документация функционала вкладок

Дата: 2026-02-15

## Контекст

Пользователь запросил максимально подробное описание функционала каждой вкладки приложения в документации. Решено создать отдельный документ по каждой вкладке.

## Изменения

Создана директория `docs/features/` со следующими файлами:

| Файл | Содержание |
|------|------------|
| README.md | Индекс со ссылками на все вкладки |
| map-tab.md | Карта: территории, события, клубы, API |
| run-tab.md | Пробежка: журнал, трекинг, детали, Swipe-to-run |
| messages-tab.md | Сообщения: подвкладки Личные/Клуб/Тренер, WebSocket |
| events-tab.md | События: список, создание, участие, check-in |
| profile-tab.md | Профиль: данные, настройки, клубы |

Каждый документ содержит:
- Назначение вкладки
- Экраны и компоненты
- Действия пользователя
- API-эндпоинты
- Навигация и маршруты
- Состояния (loading/error/empty)
- Связанные файлы кода
