# Кастомные стили карты

**Дата:** 2026-02-28

## Описание

Введён общий стиль отображения карты Yandex MapKit и подключён к экранам выбора локации и основной карты. Затем конфигурация стиля упрощена (убрана избыточная настройка point style для business tags).

## Mobile

### Новая утилита map_style

- Файл `mobile/lib/shared/utils/map_style.dart` — общая конфигурация стиля карты (MapKit).
- Стиль задаёт единый визуальный вид карты (цвета, отображение объектов) для всего приложения.

### Подключение стиля

- **LocationPickerScreen** — при создании/редактировании события карта выбора точки старта использует общий стиль.
- **MapScreen** — основная карта вкладки «Карта» использует тот же стиль.

### Рефакторинг

- Из утилиты стиля удалена избыточная конфигурация point style для business tags — упрощение кода без изменения визуального поведения для текущих сценариев.
