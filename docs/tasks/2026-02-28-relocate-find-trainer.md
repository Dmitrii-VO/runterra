# Задача: Перенос функционала «Найти тренера»

**Дата**: 2026-02-28
**Статус**: В планировании

## Описание
Перенести точку входа «Найти тренера» из раздела «Профиль» во вкладку «Сообщения -> Тренер» для улучшения пользовательского опыта (UX). Поиск должен быть доступен как в пустом состоянии (когда тренера нет), так и постоянно (для поиска новых специалистов).

## План реализации

### 1. Удаление из Профиля
- Изменить `mobile/lib/features/profile/profile_screen.dart`.
- Удалить `Card` с `ListTile(l10n.findTrainers)`.
- **Верификация**: Вкладка профиля больше не содержит кнопку поиска.

### 2. Обновление CoachTab (Пустое состояние)
- Изменить `mobile/lib/features/messages/tabs/coach_tab.dart`.
- В методе `_buildPersonalTab` заменить текстовое описание в empty state на колонку с кнопкой `ElevatedButton`.
- **Верификация**: При отсутствии тренера видна кнопка «Найти тренера», открывающая `/trainers`.

### 3. Добавление постоянной кнопки поиска
- Добавить `IconButton` с иконкой `Icons.search` в заголовок или верхнюю часть вкладки «Личные».
- **Верификация**: Кнопка видна даже если список клиентов/тренеров не пуст.

### 4. Валидация
- Запустить `flutter analyze`.
- Проверить навигацию вручную (при наличии возможности).
